# 地球村民监察互助联盟 - 项目完整开发过程记录

## 项目概述

**项目名称**: 地球村民监察互助联盟  
**核心目标**: 建立网友投票监督平台，形成就业歧视/剥削行为榜单，推动社会透明化

---

## 第一部分：初始架构设计与规划

### 1.1 技术栈选择理由

| 组件 | 技术方案 | 选择理由 |
|------|----------|----------|
| Web 框架 | Django + DRF | 内置用户认证、后台管理、ORM，适合快速开发复杂权限系统 |
| 数据库 | SQLite | 轻量级无需部署，便于本地测试和分发 |
| GUI 客户端 | PyQt6 | 功能丰富的跨平台桌面应用框架，可打包为 exe |
| 安全机制 | TokenAuthentication + 设备指纹 | 保障投票真实性和系统安全性 |

### 1.3 防刷票机制设计

- **设备指纹**: `MD5(user_agent + accept_language)`，识别浏览器环境
- **IP 频率限制**: 单 IP/日最多投票 3 次
- **重复提交检测**: 同一用户对同一企业同类别只能投一次票
- **证据材料强制关联**: 每次投票可上传证据（图片/PDF），增强可信度

## 第三部分：用户认证系统开发过程

### 3.1 CustomUser 与 Native User 分离架构

**设计决策**: 
- **CustomUser**: 存储加密手机号、验证码等敏感信息
- **Native User (django.contrib.auth.User)**: DRF TokenAuthentication 必须关联到此模型

**问题**: Vote.voter FK 指向 CustomUser，但登录返回的是 NativeUser 实例。

### 3.2 Username Bridge Pattern 实现

**views.py 关键代码段** (Line 50-60):

```python
from django.contrib.auth import get_user_model
NativeUser = get_user_model()

if isinstance(request.user, NativeUser):
    custom_user = CustomUser.objects.filter(username=request.user.username).first()
    if not custom_user:
        return Response({'error': '用户不存在'}, status=status.HTTP_404_NOT_FOUND)
else:
    custom_user = request.user
```

**实现逻辑**: 
1. 登录时，NativeUser 与 CustomUser 通过相同 username 关联
2. TokenAuthentication 返回 NativeUser 实例
3. View 中通过 `filter(username=...)` 查询对应的 CustomUser 对象
4. 投票创建时使用 CustomUser 作为 voter FK

```python
from django.db import connection

    cursor.execute('ALTER TABLE evidence ADD COLUMN file_path VARCHAR(500)')
    cursor.execute('ALTER TABLE evidence ADD COLUMN file_type VARCHAR(20)')
```

### B.1 完整端到端测试脚本 (Python Requests)

```python
import requests

# 登录获取 Token
s = requests.Session()
login_resp = s.post('http://127.0.0.1:8000/api/auth/login/', 
                    json={'username': 'testuser', 'password': 'TestPass123!'})
token = login_resp.json()['token']
s.headers['Authorization'] = f'Token {token}'

# 搜索企业
comp_resp = s.get('http://127.0.0.1:8000/api/vote/companies/search/', params={'q': '华为'})
company_id = comp_resp.json()[0]['id']

# 提交投票
vote_resp = s.post('http://127.0.0.1:8000/api/vote/submit/', 
                   data={
                       'category': 'age_discrimination',
                       'company_id': company_id,
                       'reason': '测试企业存在年龄歧视行为'
                   })

# 上传证据
evidence_resp = s.post('http://127.0.0.1:8000/api/vote/evidence/upload/', 
                       data={'vote_id': vote_resp.json()['vote_id']},
                       files={
                           'evidence_file': (
                               'proof.pdf',
                               b'PDF Content Here',
                               'application/pdf'
                           )
                       })

print(f"Vote ID: {vote_resp.json()['vote_id']}")
print(f"Evidence URL: {evidence_resp.json()['file_url']}")
```
